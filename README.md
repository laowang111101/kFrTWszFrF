# 学生成绩管理系统 python1045

## 项目概述

学生成绩管理系统是一款基于Python的Django框架和Vue.js的前端技术构建的在线系统。该系统旨在帮助学校管理学生、教师、课程及成绩信息，实现教育信息化管理。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

- **首页**：系统概览，快捷入口。
- **个人中心**：用户个人信息管理。
- **教师管理**：教师信息增删改查。
- **学生管理**：学生信息增删改查。
- **公告信息管理**：公告的发布与查询。
- **课程类型管理**：课程类型设置与修改。
- **课程信息管理**：课程详细信息的管理。
- **选课信息管理**：学生选课信息记录与管理。
- **成绩信息管理**：学生成绩录入、查询与修改。

## 系统角色

- 系统管理员
- 教师
- 学生

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Node.js
- Vue.js

## 部署步骤

1. 安装Python和MySQL环境。
2. 使用pip安装Django及依赖。
3. 克隆前端Vue项目代码，并安装依赖。
4. 配置数据库连接。
5. 运行Django后端项目。
6. 运行Vue前端项目。

## 目录结构

```
student-grade-management-system/
├── backend/              # 后端Django项目目录
├── frontend/             # 前端Vue项目目录
└── database/             # 数据库文件目录
```

## 核心亮点

- **基于Python的Django框架**：成熟稳定，易于维护和扩展。
- **前后端分离**：采用Vue.js构建前端，分离的开发模式便于分工合作。
- **模块化设计**：功能模块清晰，易于定制化开发。
